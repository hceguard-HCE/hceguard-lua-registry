# HCEGuard LUA Registry

Registro público de scripts LUA compatibles con HCEGuard.

Este repositorio ayuda a revisar, documentar y registrar scripts LUA usados en Halo Custom Edition de forma transparente y auditable.

## Objetivo

HCEGuard promueve juego limpio, seguridad y transparencia.

Los scripts LUA privados, cerrados o no verificables no serán considerados aprobados oficialmente.

## Licencias

HCEGuard no reclama propiedad sobre scripts de terceros.

Cada script conserva la licencia de su autor original.  
Si un script no tiene licencia explícita o permiso del autor, solo podrá ser referenciado, no copiado.

## Requisitos

Para ser aprobado, un script debe:

- Ser público y auditable.
- Tener autor o fuente identificable.
- No dar ventajas injustas.
- No recolectar datos de forma oculta.
- No tener comportamiento malicioso.
