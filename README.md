# HCEGuard LUA Registry

Registro público de scripts LUA compatibles con HCEGuard.

Este repositorio sirve para documentar, revisar y registrar scripts LUA usados en Halo Custom Edition, con el objetivo de mantener una comunidad más transparente, segura y auditable.

## Objetivo

HCEGuard busca promover el juego limpio y la transparencia.  
Los scripts LUA privados, cerrados o no verificables no serán considerados aprobados oficialmente.

## Licencias

HCEGuard no reclama propiedad sobre scripts creados por terceros.

Cada script conserva la licencia indicada por su autor original.  
Si un script no tiene licencia explícita, no se considera de uso libre y solo podrá ser referenciado o revisado si existe autorización del autor.

La documentación, políticas y archivos de metadata creados por HCEGuard pueden publicarse bajo licencia MIT, salvo que se indique lo contrario.

## Requisitos para aprobación

Para que un script LUA sea considerado compatible con HCEGuard, debe cumplir:

- Código público y auditable.
- Repositorio accesible, por ejemplo GitHub, GitLab o similar.
- Hash SHA256 registrado.
- Autor o fuente identificable.
- Sin comportamiento malicioso.
- Sin ventajas injustas para el jugador.
- Sin recolección oculta de datos.

## Estados

- `approved`: aprobado oficialmente.
- `pending_review`: pendiente de revisión pública.
- `unverified`: no verificado.
- `rejected`: rechazado.
- `blocked`: bloqueado por comportamiento malicioso.

## Nota

Que un script aparezca en este repositorio no significa que HCEGuard sea dueño del script.  
Este repositorio funciona como registro de compatibilidad, revisión y transparencia.
